> # **login system**

it is a small app simulating login , sign up and log out

> ## skills

- html
- css
- bootstrap
- motion

> ## decription

when openning the app site for first time , login page will ask you to enter
your email and password if you register before in the site else you can click on sign up
link to register to a new user ,after you has register and entered correct info , then you will be redirected to the home page and if you close the tap or the broswer ,and open it after a while you will still loged in
,and you can also log out.

> ## features

- dark mode.
- banners to help user.
- animations.
